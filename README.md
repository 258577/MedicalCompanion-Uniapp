# Hospitaluniapp

一个基于uni-app开发的医疗服务微信小程序，用于练习前端开发和学习微信小程序开发技术。

## 项目介绍

该项目是一个医疗服务类微信小程序，提供医院查询、服务预约、订单管理等功能，旨在为用户提供便捷的医疗服务信息查询和预约体验。

## 技术栈

- **前端框架**：uni-app
- **开发语言**：Vue.js
- **UI组件**：自定义组件 + uni-ui
- **开发工具**：HBuilderX
- **目标平台**：微信小程序

## 功能模块

1. **首页**
   - 轮播图展示
   - 快捷功能入口
   - 医院列表展示
   - 医院搜索功能

2. **医院详情**
   - 医院基本信息展示
   - 医院服务介绍

3. **订单管理**
   - 订单列表
   - 订单详情

4. **服务预约**
   - 填写服务订单

5. **用户中心**
   - 个人信息管理

## 项目结构

```
├── common/           # 公共工具类
├── components/       # 通用组件
├── pages/            # 页面文件
│   ├── index/        # 首页
│   ├── hospital/     # 医院详情
│   ├── order/        # 订单管理
│   ├── service/      # 服务预约
│   ├── user/         # 用户中心
│   └── search/       # 搜索页面
├── static/           # 静态资源
├── uni_modules/      # uni-app插件
├── App.vue           # 应用入口文件
├── main.js           # 应用入口配置
├── manifest.json     # 应用配置
├── pages.json        # 页面路由配置
└── uni.scss          # 全局样式
```

## 安装和运行

### 环境要求

- HBuilderX
- Node.js
- 微信开发者工具

### 安装步骤

1. 克隆项目到本地
   ```bash
   git clone <仓库地址>
   ```

2. 使用HBuilderX打开项目

3. 安装依赖
   ```bash
   npm install
   ```

4. 运行项目
   - 选择微信小程序运行环境
   - 使用微信开发者工具导入项目

## 开发工具

- **HBuilderX**：用于uni-app项目开发
- **微信开发者工具**：用于微信小程序调试和预览
